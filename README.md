# CodeNexus Academy

A beautiful, interactive website for learning web development fundamentals - HTML, CSS, and JavaScript. CodeNexus Academy provides hands-on learning with live code examples.

## Features

- **Interactive Learning**: Learn HTML, CSS, and JavaScript through hands-on examples
- **Live Code Editor**: Edit and run code directly in the browser
- **Beautiful UI**: Modern, responsive design with smooth animations
- **Comprehensive Examples**: Multiple examples for each topic
- **Mobile Responsive**: Works perfectly on all devices

## Getting Started

Simply open `index.html` in your web browser to start learning!

## Sections

### HTML Basics
- Learn HTML structure and elements
- Interactive examples with live preview
- Common HTML elements explained

### CSS Basics
- Style your web pages
- Learn CSS selectors and properties
- Flexbox, animations, and more

### JavaScript Basics
- Add interactivity to your pages
- DOM manipulation
- Event handling
- Functions and variables

## Technologies Used

- HTML5
- CSS3 (with modern features like Flexbox, Animations)
- Vanilla JavaScript (no frameworks required)
