HTML Basics

Learn the foundation of web development with HTML (HyperText Markup Language)

What is HTML?

HTML is the standard markup language for creating web pages. It describes the structure of a web page and consists of a series of elements that tell the browser how to display content.

Basic HTML Structure

<!DOCTYPE html>
<html lang="en">
<head>
    <title>My First Page</title>
</head>
<body>
    <h1>Hello World</h1>
</body>
</html>

Common HTML Elements

  • <h1> to <h6> - Headings
  • <p> - Paragraphs
  • <a> - Links
  • <img> - Images
  • <div> - Container
  • <span> - Inline container
  • <button> - Buttons
  • <input> - Form inputs

CSS Basics

Style your web pages with CSS (Cascading Style Sheets) to create beautiful designs

What is CSS?

CSS is used to style and layout web pages. It allows you to control colors, fonts, spacing, positioning, and much more.

CSS Syntax

selector {
    property: value;
    property: value;
}

CSS Selectors

  • element - Selects all elements of that type
  • .class - Selects elements with that class
  • #id - Selects element with that ID
  • element.class - Combines element and class

Common CSS Properties

  • color - Text color
  • background-color - Background color
  • font-size - Text size
  • margin - Outer spacing
  • padding - Inner spacing
  • border - Border styling
  • width / height - Dimensions

JavaScript Basics

Add interactivity and dynamic behavior to your web pages with JavaScript

What is JavaScript?

JavaScript is a programming language that enables interactive web pages. It's an essential part of web development alongside HTML and CSS.

Basic JavaScript Syntax

// Variables
let name = "CodeNexus";
const age = 2024;

// Functions
function greet() {
    console.log("Hello!");
}

// Event Listeners
button.addEventListener('click', function() {
    alert('Clicked!');
});

JavaScript Concepts

  • Variables - Store data (let, const, var)
  • Functions - Reusable code blocks
  • Events - User interactions
  • DOM - Manipulate HTML elements
  • Arrays - Store multiple values
  • Objects - Store key-value pairs
  • Loops - Repeat code execution
  • Conditionals - Make decisions
